### 架构设计

> 难点：Pyscript 和 JS 的交互需要触发事件；没有数据传输通道

触发流程

- 点击 button-run
	- 检查当前是否focus在某个单元格上
	- 若在，则将当前单元格的值赋值给 #input
	- 触发 input 的 click 事件
- #input 上绑定了 python 代码中的某个指定函数
	- click 触发该函数
	- 从 #input 的值中获取单元格的输入
	- 扔给后端计算
	- 将结果 str() 后，赋值给 #output
	- 触发 #output 的 click 事件
- #output 上绑定了 click 事件
	- 从自身取值，填入单元格中
